# Windows Administration Scripts

A collection of Windows batch, PowerShell, and registry utilities designed to simplify common administration, troubleshooting, and maintenance tasks.

## Included Scripts

### Context Menu Utility

Adds useful shortcuts and tools to the Windows context menu.

### Android WSA Icon Fix

PowerShell utility for repairing missing Windows Subsystem for Android icons.

### Kill Applications

Terminates selected applications and background processes.

### Printer Troubleshooting

Automates common printer diagnostics and recovery steps.

### Wi-Fi Troubleshooting

Resets network components and performs connectivity diagnostics.

### Shell Commands Helper

Provides quick access to commonly used Windows shell commands.

## Technologies

* Windows Batch
* PowerShell
* Windows Registry

## Requirements

* Windows 10 or Windows 11
* Administrator privileges for some scripts

## Usage

Run the desired script from an elevated Command Prompt or PowerShell session where required.

## Disclaimer

Review each script before execution and ensure it is suitable for your environment.
